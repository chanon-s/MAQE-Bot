# MAQE-Bot

## Minimum system requirements

NodeJS version 18 or above.

## Build
```bash
npm install
npm run build
```

## How to Run

```bash
node index.js RW15RW1
```
> Change RW15RW1 to your command

## Source Code

In this project I create by Typescript and then build to Javascript.
